## Digit Recognizer: CNN Solution for Kaggle Competition
Решение для соревнования Kaggle «Digit Recognizer» — классификация рукописных цифр (0–9) по изображениям 28 × 28 пикселей.

## Результат
Score на публичном лидерборде: 0.99564 (accuracy)

Позиция в соревновании: 113‑е место из 1 440 команд

Метрика: accuracy (доля правильных ответов)

Описание подхода
Решение сочетает:

K‑Nearest Neighbors (KNN) для быстрого базового результата.

Свёрточную нейронную сеть (CNN) с аугментацией данных для достижения высокого качества.

Ключевые компоненты
1. Предобработка данных
Нормализация пикселей: [0, 255] → [0, 1].

Изменение формы: (N, 784) → (N, 28, 28, 1) для CNN.

Разделение на train/val: 80 % / 20 % с сохранением баланса классов (stratify=y_train).

One‑hot кодирование меток (to_categorical).

2. Модель KNN (базовый уровень)
n_neighbors=3.

Обучение на подвыборке (10 000 примеров для скорости).

Предсказания на полном тестовом наборе.

3. CNN‑модель (финальное решение)
Архитектура:

3 свёрточных блока (Conv2D + BatchNormalization + MaxPooling + Dropout).

Полносвязный слой (512 нейронов, Dropout = 0.5).

Выход: 10 классов (softmax).

Обучение:

Оптимизатор: Adam (lr=0.001).

Функция потерь: categorical_crossentropy.

Аугментация: повороты, сдвиги, масштабирование (ImageDataGenerator).

Callback‑и:

ReduceLROnPlateau (снижение LR при застое).

EarlyStopping (остановка при переобучении).

Эпохи: до 50 (с ранней остановкой).

Batch size: 32.
